## Executive Overview

Graphwise faces a critical market positioning challenge in a rapidly growing semantic web market (projected $7.1B to $48.4B by 2030, 37.8% CAGR). Despite having a unique end-to-end Graph AI Suite combining GraphDB and PoolParty, **no standardized benchmarking methodology exists** for comparing semantic layer platforms across RDF triple stores and property graph systems.

This project delivers a **20-week academic-quality benchmarking study** that establishes standardized performance metrics across 6 major semantic platforms, positioning Graphwise as the thought leader in hybrid AI (deterministic + probabilistic) while providing actionable competitive intelligence for FY2026 planning.

### Key Highlights

- **87% team match** combining academic excellence (IEEE Best Paper 2023) with 14+ years enterprise experience
- **6 platforms evaluated**: Graphwise, Neo4j, Stardog, AWS Neptune, TigerGraph, MongoDB
- **Academic rigor**: Publication-ready for ISWC/ESWC conferences
- **GraphRAG focus**: Extending data.world's methodology (238% LLM accuracy improvement)
- **$125K+ consulting value** delivered through university partnership

## Project At-A-Glance

### Client Profile
- **Company**: Graphwise (October 2024 merger: Ontotext GraphDB + Semantic Web Company PoolParty)
- **Employees**: ~200 across North America, Europe, APAC
- **Key Clients**: AstraZeneca, BBC, Ernst & Young, Johnson & Johnson, NASA, World Bank
- **Unique Position**: Only end-to-end Graph AI Suite with RDF reasoning + semantic metadata

### Project Scope
- **Duration**: 20 weeks (January - May 2026)
- **Budget Equivalent**: $125,000+ consulting value
- **Platforms Evaluated**:
  - RDF Systems: Graphwise GraphDB, Stardog, Franz Inc.
  - Property Graphs: Neo4j, TigerGraph, AWS Neptune
  - Hybrid: MongoDB (document + graph)

### Key Deliverables
1. **Academic Research Paper** (25-35 pages, peer-review ready)
2. **Benchmark KPI Framework** (performance, scalability, GraphRAG, TCO)
3. **Competitive Intelligence Report** (6-vendor comparison)
4. **Open Source Benchmark Suite** (datasets, queries, scripts)

### Team Composition

  - Founded AaMaRa Technologies, 15+ Fortune 500 clients
- **Technical Architect** (92% match)
  - IEEE Best Paper 2023, AWS DynamoDB, Morgan Stanley
- **Data Engineering Lead** (85% match)
  - AWS/Azure expert, 35% data ingestion improvement
